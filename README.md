

Terraform 실행은 아래 명령어 순서대로 진행하시면 됩니다.


Terraform 환경 설정
````
terraform init # Terraform 환경 설정
````
Terraform 배포할 리소스들 확인
````
terraform plan # Terraform 배포할 리소스들 확인
````
Terraform 리스소 생성
````
terraform apply # Terraform 리스소 생성
````
